# Underscore.ext

Lightweight JavaScript utilities for mobile web, based on Underscore and Zepto.

专注移动端的 Javascript 工具库，基于 Underscore 和 Zepto。

***

## 构建过程

本项目采用Gulp实现项目自动化构建

0. 安装Gulp
	```sh
	$ npm install -g gulp   
	$ npm install --save-dev gulp    
	```

1. 安装依赖插件
	```sh
	$ npm install --save-dev gulp-concat gulp-uglifyjs del
	```

2. 运行Gulp
 	```sh
	$ gulp
	```
